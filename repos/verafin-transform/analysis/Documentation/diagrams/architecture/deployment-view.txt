DEPLOYMENT CHARACTERISTICS:
===========================
Module Structure:  Single JAR (monolithic)
Runtime:           Java 11 JVM
Memory:            Heap size depends on JVM settings
Threads:           Single or multi-threaded (not specified)
Configuration:     System properties (not externalized)
Database Driver:   PostgreSQL JDBC (implied)

DEPLOYMENT TOPOLOGY:
====================
Tier 1: Application (JVM Process)
Tier 2: Database (PostgreSQL Server)

CONNECTION:
===========
- Protocol: JDBC over TCP/IP
- Port: 5432 (PostgreSQL default)
- Authentication: Username/Password
- Connection Pooling: Not implemented

SCALABILITY:
============
- Horizontal: Requires external load balancer
- Vertical: Limited by JVM heap size
- Database: Single instance (no clustering)
